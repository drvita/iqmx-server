# IQISSMexico (iqmx) - Web

Este repositorio contiene el código fuente para el sitio web corporativo de **IQISSMexico** y sus landing pages de productos.

## Descripción del Proyecto

IQISSMexico es una empresa dedicada al desarrollo de software y soluciones tecnológicas. Este proyecto web está construido con [Next.js](https://nextjs.org/) y [Tailwind CSS](https://tailwindcss.com/) para ofrecer una experiencia rápida, moderna y escalable.

### Estructura del Sitio

- **/**: Página principal corporativa. Información sobre la empresa, servicios y contacto.
- **/landingpage/whatsapp**: Landing page dedicada al producto de automatización de WhatsApp.

## Tecnologías Utilizadas

- **Framework**: Next.js 15 (App Router)
- **Estilos**: Tailwind CSS
- **Lenguaje**: TypeScript

## Comenzando

Para ejecutar el servidor de desarrollo localmente:

```bash
npm run dev
# o
yarn dev
# o
pnpm dev
# o
bun dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver el resultado.

## Despliegue

El proyecto está optimizado para ser desplegado en [Vercel](https://vercel.com/new?utm_medium=default-template&filter=next.js&utm_source=create-next-app&utm_campaign=create-next-app-readme), la plataforma de los creadores de Next.js.

Consulta la [documentación de despliegue de Next.js](https://nextjs.org/docs/app/building-your-application/deploying) para más detalles.
